Файлы графических ресурсов V*.MCT игры MAGIC CANDLE 2 содержат тайлы карты.
Они устроены так:
+000h DB*768 палитра, сдвинутая на 2 бита влево
+300h DB*256*? тело файла
Тело файла представляет собой набор тайлов 16*16, каждый из которых представляет
собой простой битмап. Файл содержит 128 или 256 таких тайлов.

Файл графических ресурсов V.MCT игр MAGIC CANDLE 3 и BLOODSTONE состоит из
заголовка и набора модулей - файлов вышеописанной структуры. Его заголовок
устроен так:
+000 DD смещение в файле 1-го модуля (0 если этот модуль пропущен)
+004 DD смещение в файле следующего за его концом байта (-"-)
+008 DD смещение в файле 2-го модуля (-"-)
+00C DW смещение в файле следующего за его концом байта (-"-)
+010 ... так же для других модулей
+320 ... первый модуль
Единственное отличие файлов-модулей от вышеописанного формата отдельных файлов
в том, что все они содержат по 240 тайлов.
