Файл графики .DFA игры THE ELDER SCROLLS: THE ARENA состоит из заголовка,
базовой страницы и набора дополнительных страниц изображения.Заголовок файла
устроен так:
+00 DW число дополнительных страниц
+02 DW ? (похоже,не используется)
+04 DW ? (-"-)
+06 DW ширина картинки
+08 DW высота картинки
+0A ... базовая страница
+?? ... 1-я дополнительная страница
Базовая страница устроена так:
+0 DW размер-2
+2 ... тело страницы
Тело базовой страницы представляет собой простой битмап,сжатый блочным методом:
 блок копирования: DB 0xxxxxxx / DB (X+1) байт;
 блок заполнения:  DB 1xxxxxxx / DB байт; этот байт повторяется (X+1) раз;
0 - прозрачный цвет.
Дополнительные страницы выводятся каждая поверх базовой. Они устроены так:
+0 DW размер-2
+2 DW число блоков
+4 ... блоки
Блоки дополнительной страницы устроены так:
+0 DW смещение от начала картинки (левого верхнего угла) в байтах
+2 DW число точек в блоке
+4 DB*? сами точки.
Последняя дополнительная страница часто бывает фиктивной (0 блоков).
