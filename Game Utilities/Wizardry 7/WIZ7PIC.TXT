Файл графики .PIC в Wizardry 7 состоит из заголовка и набора картинок.Картинки
состоят из заголовка и набора блоков.Блоки представляют собой байтовые массивы
8*8, прозрачный цвет 0FFh.
Заголовок файла устроен так:
+0 DD размер файла-4
+4 DW число картинок
+6 DW смещение тела картинки-4
+8 DW сегмент тела картинки (от начала файла)
+A ... (так же для всех картинок)
Заголовок картинки устроен так:
+0 DW размер заголовка
+2 DB число блоков по горизонтали
+3 DB число блоков по вертикали
+4 DB*? битовая карта: для каждого блока по порядку (слева направо и сверху
   вниз) бит с соответствующим номером равен 1, если блок не пустой, 0 - если
   пустой (сплошь прозрачный цвет)
+? ... тела всех _непустых_ блоков по порядку
