Unpacker v0.1 (24/10/2009)
Первая версия. Распаковываются только те файлы, которые в текущий момент нужны игре для обработки

Unpacker v0.11 (26/10/2009)
Исправлена небольшая ошибка с неправильным определением размера распакованного файла, в результате чего файлы распаковывались не до конца
Формат файлов english_NN.out:
#C - этот тег центрирует надпись (заголовки книг)
#P - переход на новую страницу книги
#T - новый абзац
#G - print number (пока не понял что это, возможно номер страницы? Или еще какое-то число)
 - символ конца файла

Unpacker v0.12 (26/10/2009)
Распаковщик научился полностью распаковывать языковые паки. Поддержка остальных архивов пока ограничена

Unpacker v0.2 (02/11/2009)
Удалил ненужный распаковщику код движка игры, размер программы уменьшился где-то в 6 (!) раз и составляет теперь несчастные 77 КБ. Теперь распаковщик запускается без ненужной загрузки игры, а главное - теперь можно ввести имя любого pak-файла для распаковки и он полностью распакуется! Теперь должны распаковываться все pak-файлы, а не только english и itd_ress.
Также разобрался, почему предыдущие релизы не запускались ни у кого, кроме меня))) Теперь должно работать у всех

Unpacker v0.3 (03/11/2009)
Добавлена утилита для обратной запаковки pak-файлов. Лучше пользоваться этими программами в связке - сначала распакуйте нужные pak-файлы, потом модифицируйте распакованные файлы и запакуйте их упаковщиком. Для запаковки необходимо наличие ВСЕХ распакованных файлов из пака. Желательно наличие и исходного pak-файла. Упаковка происходит без сжатия (размер пака увеличивается в несколько раз, но игра такие файлы читает без проблем, проверялось на english.pak).

Unpacker v0.4 (13/11/2009)
Добавлена поддержка немного нестандартных паков из игры Jack in the dark. Теперь они тоже должны распаковываться нормально.
Также добавлен упаковщик отдельных файлов res_packer (предыдущая версия упаковщика могла перепаковывать только все файлы из пака, по отдельности никак). Утилита была написана прежде всего для упаковки шрифта с поддержкой кириллицы. Без нее пак разрастался до неимоверных размеров (около 1 Мб), и игра переставала работать.
Настоятельно рекомендуется использовать утилиту для упаковки небольшого числа файлов, поскольку при упаковке большого числа файлов размер пака будет сильно увеличиваться. Для упаковки всех файлов пользуйтесь старой версией упаковщика.
