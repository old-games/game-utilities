Файл графики .STP в игре PROJECT NOMAD состоит из заголовка и тела. Тело состоит
из набора блоков.
Заголовок файла устроен так:
+0 DW число столбцов
+2 DW число строк
+4 DW собственный сдвиг слева(?) (обычно 0)
+6 DW собственный сдвиг сверху(?) (-"-)
+8 ... тело файла
Блоки устроены так:
блок пропуска:
+0 DB 1xxxxxxx, по нему пропускается (xxxxxxx) точек (прозрачный цвет)
блок заполнения:
+0 DB 01xxxxxx
+1 DB цвет , по нему (xxxxxx) точек закрашиваются этим цветом
блок копирования:
+0 DB 00xxxxxx
+1 DB (xxxxxx) байт изображения, по нему они выводятся на экран
