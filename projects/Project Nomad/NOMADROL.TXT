Файл графики .ROL в игре PROJECT NOMAD состоит из заголовка и набора картинок.
Картинки, представляющие собой .STP-файлы, состоят из заголовка и тела. Тело
картинки состоит из набора блоков.
Заголовок файла устроен так:
+0 DD смещение в файле 1-й картинки (оно же размер заголовка)
+4 DD смещение 2-й картинки
+8 ... так же для остальных картинок
Заголовок картинки устроен так же как заголовок .STP-файла:
+0 DW число столбцов
+2 DW число строк
+4 DD 0
+8 ... тело картинки
Блоки устроены так:
блок пропуска:
+0 DB 1xxxxxxx, по нему пропускается (xxxxxxx) точек (прозрачный цвет)
блок заполнения:
+0 DB 01xxxxxx
+1 DB цвет , по нему (xxxxxx) точек закрашиваются этим цветом
блок копирования:
+0 DB 00xxxxxx
+1 DB (xxxxxx) байт изображения, по нему они выводятся на экран
