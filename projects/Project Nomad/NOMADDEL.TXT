Файл графики .DEL в игре PROJECT NOMAD состоит из заголовка и тела. Тело состоит
из набора блоков.
Заголовок файла устроен так:
+0 DW число столбцов
+2 DW число строк
+4 ... тело файла
Блоки устроены так:
короткий блок пропуска:
+0 DB xxxxxx10, (xxxxxx)>0, по нему пропускается (xxxxxx) точек (прозрачный
      цвет)
длинный блок пропуска:
+0 DB 00000010
+1 DB cnt, по нему пропускается (cnt) точек (прозрачный цвет)
блок заполнения:
+0 DB xxxxxx01
+1 DB цвет , по нему (xxxxxx) точек закрашиваются этим цветом
блок копирования точки:
+0 DB xxxxxx11
+1 DB байт изображения, по нему он выводится на экран
блок разностного сжатия:
+0 DB xxxxxx00
+1 DB цвет С[0]
+2 DB (xxxxxx/2) байт данных, по нему (xxxxxx) точек закрашиваются по правилу:
      С[n+1]=С[n]+d, где d зависит от n-й тетрады данных так: 0..7 - 0..7;
      8..F - -8..-1 (тетрады выбираются с младших адресов, старшая тетрада перед
      младшей)
