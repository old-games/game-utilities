Файл графики .ART в игре DUKE NUKEM 3D состоит из заголовка и набора спрайтов.
Заголовок файла устроен так:
+0  DD     версия (всегда 1)
+4  DD     общее число картинок во всех ART-файлах игры вместе
+8  DD     порядковый номер первой картинки в файле в сквозной нумерации
+C  DD     то же для последней картинки
+10 DW*?   ширина для каждой картинки
+?  DW*?   высота для каждой картинки
+?  DB*4*? данные анимации для каждой картинки
+?  ...    собственно тела картинок, идущие подряд
Данные анимации устроены так:
+0 DB xxyyyyyy - здесь yyyyyy - число фаз анимации-1, а xx - порядок смены
      фаз: 00 - анимации нет, 01 - осцилляция (вперед-назад), 10 - вперед, 11 -
      назад; этот байт не равен 0 только для первой из фаз в последовательности
+1 DB смещение по X точки вставки (со знаком)
+2 DB то же, по Y
+3 DB скорость анимации (так же не 0 только для первой из фаз анимации)

Тело картинки - простой битмап, хранящийся по _столбцам_; 0FFh - прозрачный
цвет.
